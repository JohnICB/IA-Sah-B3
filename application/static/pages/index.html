Forsyth-Edwards Notation (FEN) is a standard notation for describing a particular board position of a Chess game.

The purpose of FEN notation is to provide all the necessary information to restart a game from a particular position.

Help center

This application allows users to play chess in an easy way by offering the participants posibilities to see the next move using some specific strategies depending on the current configuration of the chess table.

How can we see the next move?

"Get suggested moves" BUTTON shows us a series of moves computed by some algorithms that analyse the best move related to the current possition of the table.

How can we flip the board?

We can flip the board by pressing the "Flip board" BUTTON.

How can we set the interactive board in a specific state?

We can set the interactive board by introducing in the "FEN " INPUT a state of table and then press the "Submit" BUTTON.

How can we see the strategies details?

In the navigation bar you can use the "Strategies details" BUTTON, which presents the information every strategy that has been used in the game.

What contains "Match analysis"?

The "Match analysis" page contains a interactive table, a component with the following zones: game history and game controller(play, pause, undo, reset).
