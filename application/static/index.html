Forsyth-Edwards Notation (FEN) is a standard notation for describing a particular board position of a Chess game.

The purpose of FEN notation is to provide all the necessary information to restart a game from a particular position.

Suggested Moves

The suggested moves component can be opened by clicking on the green Suggested moves link. From the expanded area you can choose to do one of the following:

Get suggested moves
Clicking the button will trigger all the strategies & algorithms to suggest a best move, given the current state of the board. Hovering over any of the strategies highlights the move on the chessboard.
Flip the chessboard
Flip the board button literally flips the table so the opposite color is in front of you.
Set the chessboard FEN
If you want to set the chessboard to a specific configuration, you can use the Set chessboard FEN field. You can learn more about the FEN notation here.
See details about a strategy
Except from the short description found right under the name of the strategy/algorithm, clicking on any strategy will bring up an element with a more detailed overview and a link to some documentation.
Commentary

The commentary component can be opened by clicking the Commentary blue link. Said component produces a summary of a finished match exposing key points in a match where the platform computed a better move. The computed moves are highlighted with the color blue, while the user's move are highlighted with green. Clicking either the blue or green move from a commentary line will set the chessboard configuration to the on right before that move. Hovering over the moves highlights them on the chessboard but only after a click happened.

In order to achieve the finished state you can do one of two things:

  1. Play using the interactive chessboard until checkmate. In this scenario the app will automatically generate a summary.
  2. Using the match loader field submit a string that represents a sequence of (from, to)moves. The string must represent a match from start until checkmate. Press the Submit button to get the summary.

Current FEN

The current FEN (configuration) is right under the chessboard it's always updating as a reaction to any move that takes place. Right next to it there is Copy to clipboard button, in case you want to save the current state.
